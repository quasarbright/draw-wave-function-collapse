Draw a seed. Weave a world.

Paint a tiny pattern and overlapping wave function collapse grows it into a much larger world — every neighborhood in the output echoes something from your seed.

Overlapping wave function collapse

The algorithm never invents new tiles — it only ever reassembles pieces it already saw in your seed, so the output always feels like it belongs to the same world.

Learning

It slides an N×N window over your seed (wrapping around the edges) and records every distinct little pattern it finds, plus how often each one occurs. With symmetries on, it also records each pattern's rotations and reflections — a seed with one tree can still produce a forest with trees facing every direction.

Weaving

The output starts as a grid of undecided cells, each able to become any pattern seen during learning. Repeatedly, it collapses the most constrained cell — the one with the fewest remaining options — into a single pattern, chosen randomly in proportion to how often that pattern appeared in your seed. That choice ripples outward: neighboring cells lose any pattern that no longer overlaps consistently.

When it fails

Sometimes a cell runs out of options entirely — a contradiction. When that happens the loom just starts over; it retries a number of times before giving up and asking you to loosen the seed a little (a smaller pattern size or symmetries on both help).
